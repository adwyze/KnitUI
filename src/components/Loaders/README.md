# Loaders

## Usage

```javascript
import { IconLoader } from "KnitUI"

return <IconLoader size="2rem" strokeWidth="2" fill="black" />
```

| Prop name   | Type   | Optional | Default     |
| ----------- | ------ | -------- | ----------- |
| size        | string | Yes      | "1.8rem"    |
| strokeWidth | number | Yes      | 4           |
| fill        | string | Yes      | "Neutral90" |
