# Loaders

## Usage

```javascript
import { IconLoader } from "KnitUI"

return <IconLoader size="2rem" strokeWidth="2" />
```

| Prop name   | Type   | Optional | Default  |
| ----------- | ------ | -------- | -------- |
| size        | string | Yes      | "1.8rem" |
| strokeWidth | string | Yes      | 4        |
